SYSTEM INSTRUCTION (GLOBAL):
All pages in this project MUST strictly follow modern SEO best practices.

This project is for "Lords Skill Academy (LSA)" — a skill-first, career-focused education academy.

ALL code and content must satisfy the following 10 pillars:

--------------------------------------------------
1. SEO OPTIMIZED
--------------------------------------------------
- Every page must have a unique, keyword-focused <title>
- Every page must have a unique <meta name="description">
- Keywords must be used naturally (no stuffing)
- Headings must clearly describe page intent

--------------------------------------------------
2. MOBILE-FRIENDLY / MOBILE OPTIMIZED
--------------------------------------------------
- Mobile-first CSS
- Use responsive units (%, rem, vw)
- Use Flexbox & Grid only
- Touch-friendly buttons and links
- No horizontal overflow on mobile

--------------------------------------------------
3. USER-FRIENDLY (UX)
--------------------------------------------------
- Clear navigation
- Logical content flow
- Readable font sizes
- Consistent spacing
- Simple language for students & parents

--------------------------------------------------
4. CRAWLABILITY
--------------------------------------------------
- Use semantic HTML5 elements
- Avoid hidden content that search engines cannot read
- Use clean anchor links (<a href>)
- No JavaScript-only content rendering

--------------------------------------------------
5. INDEXABILITY
--------------------------------------------------
- No "noindex" or "nofollow" unless explicitly required
- Proper meta robots tag
- Clean URLs (no parameters)
- Include canonical tags if needed

--------------------------------------------------
6. WHITE HAT SEO ONLY
--------------------------------------------------
- No fake claims
- No keyword stuffing
- No cloaking
- No deceptive redirects
- Honest, transparent content

--------------------------------------------------
7. QUALITY CONTENT
--------------------------------------------------
- Content must be:
  - Original
  - Helpful
  - Student-first
  - Career-focused
- Avoid generic coaching-class language
- Explain WHY, not just WHAT

--------------------------------------------------
8. TECHNICAL SEO
--------------------------------------------------
- Fast loading pages
- Optimized images (alt, width, height)
- Deferred JavaScript where applicable
- No render-blocking CSS
- Clean, valid HTML

--------------------------------------------------
9. ON-PAGE SEO
--------------------------------------------------
- One <h1> per page
- Proper heading hierarchy (h1 → h2 → h3)
- Internal linking between relevant sections/pages
- Descriptive anchor text
- Optimized images and media

--------------------------------------------------
10. OFF-PAGE SEO (CONTENT SUPPORT)
--------------------------------------------------
- Content must be link-worthy
- Include sections suitable for:
  - Blog linking
  - YouTube embedding
  - Social sharing
- Brand mentions must be consistent:
  "Lords Skill Academy (LSA)"
  "Affiliated with Lords Institute of Engineering & Technology (LIET)"

--------------------------------------------------
GLOBAL CONTENT RULES
--------------------------------------------------
- Use professional, confident tone
- Avoid exaggerated marketing language
- Emphasize skills, projects, outcomes
- No salary guarantees or false placement promises

--------------------------------------------------
GLOBAL TECH RULES
--------------------------------------------------
- Use only HTML, CSS, Vanilla JavaScript
- No frameworks or libraries
- Fully responsive
- Accessible (WCAG-friendly basics)

--------------------------------------------------
FINAL EXPECTATION
--------------------------------------------------
Every page must be:
- Search-engine readable
- Mobile-first
